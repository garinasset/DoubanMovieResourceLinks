# DoubanMovieResourceLinks

在豆瓣电影页面添加资源搜索入口的篡改猴脚本：

- IMDb → 海盗湾 The Pirate Bay
- 中文名 → 电影天堂（POST + GBK）

## 功能

- 自动提取 IMDb ID
- 自动获取最干净的中文片名（title）
- 支持豆瓣 SPA 页面（无需刷新）
- 无广告 / 无统计 / 无后台请求

## 安装

1. 安装 篡改猴（Tampermonkey）
2. 打开：
   https://greasyfork.org/scripts/xxxx
3. 点击安装
