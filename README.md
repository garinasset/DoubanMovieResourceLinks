# DoubanMovieResourceLinks

一个用于 **豆瓣电影页面** 的篡改猴脚本，在影片信息区自动添加第三方资源搜索链接。

---

## 功能说明

- 在 IMDb 信息下方添加资源搜索入口
- 基于 IMDb ID 搜索海外资源
- 基于中文片名搜索国内资源
- 自动识别页面，无需手动操作
- 不收集任何用户数据

---

## 支持的网站

- 海盗湾（The Pirate Bay）
- 电影天堂（Dytt）

---

## 使用方法

1. 安装 Tampermonkey（或其他用户脚本管理器）
2. 从 GreasyFork 安装本脚本
3. 打开任意豆瓣电影页面即可生效
